# Global Sales Performance Dashboard – Tableau Project

This project analyzes global retail sales using the Global Superstore dataset, a widely used open dataset for business analytics.  
The goal of the dashboard is to explore overall performance metrics, regional trends, and product-level insights through interactive visuals.

---

##  Dataset
**Source:** Global Superstore (public dataset used on Kaggle & Tableau community)

### **Key fields used**
- Sales  
- Profit  
- Order ID  
- Order Date  
- Country  
- Category / Sub-Category  
- Customer Segment  
- Region  

---

##  Tableau Visualizations
The Tableau workbook **Global Sales Performance.twb** includes the following KPIs & dashboard components:

### **KPIs**
- **Total Sales**
- **Total Profit**
- **Total Orders**

### **Charts**
- **Sales by Product Category** (horizontal bar chart)  
- **Sales by Country** (bar chart with scroll)  
- **Monthly Sales Trend** (line chart)  
- **Interactive Country Filter**

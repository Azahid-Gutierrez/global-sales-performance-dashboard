Global Sales Performance Dashboard – Tableau Project

This project analyzes global retail sales using the Global Superstore dataset, a widely used open dataset for business analytics.
The goal of the dashboard is to explore overall performance metrics, regional trends, and product-level insights through interactive visuals.

Dataset

Source: Global Superstore (public dataset used on Kaggle & Tableau community)

Key fields used:

Sales

Profit

Order ID

Order Date

Country

Category / Sub-Category

Customer Segment

Region

Tableau Visualizations

The Tableau workbook (Global Sales Performance.twb) includes the following KPIs & dashboards:

KPIs

Total Sales

Total Profit

Total Orders

Charts

Sales by Product Category (horizontal bar chart)

Sales by Country (bar chart with scroll)

Monthly Sales Trend (line chart)

Interactive Country Filter to update the entire dashboard

Dashboard Features

Clean, business-style layout

Currency formatting for Sales & Profit

Dynamic filters

Consistent styling and labels

What I Did

Imported the Global Superstore dataset into Tableau

Built 3 KPI cards to summarize key performance indicators

Created visualizations for category sales, country comparison, and monthly trends

Formatted currency values for clarity

Added an interactive Country filter to allow users to drill down

Designed and assembled the final Global Sales Performance Dashboard
